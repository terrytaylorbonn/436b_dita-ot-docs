Removing plug-ins

Use the dita uninstall subcommand to remove a plug-in.

At the command-line prompt, enter the following command: 
dita uninstall <plug-in-id>

where:

  • <plug-in-id> is the unique ID of the plug-in, as defined in the plug-inâ€™s configuration file (plugin.xml).
Attention:
The uninstall subcommand also removes the corresponding plug-in directory from the plugins folder.
